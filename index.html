Semantic HTML Tag Reference

Tag Name Description
<header> Header Defines the header of a document or section, typically containing introductory content, headings logos, and navigation links.
<nav> Navigation Dedines a section of a document dedicated to navigation links.
<main> Main Defines the main content of a document, excluding navigation, footers and sidebars.
<article> Article Defines an independant, self-contained piece of content, such as a blog post or news article
<section> Section Defines a thematic grouping of content, typically with a heading
<aside> Aside Defines the content that is tangentially related to the content around it, such as sidebar or pull quotes
<footer> Footer Defines the footer of a document or section, typically containing authorship information, copyright details, and links to related documents.
<address> Addresss Defines contact information for the author or owner of a document.
<h1>to<h6> Heading Level 1 to Heading Level 6 Defines headings of varying importance, with <h1> being the most important and <h6> beling the least important.
<p> Paragraph Defines a paragraph fo text.
<ul> Unordered List Defines an unordered list, a list of items that do not have specific order.
<ol> Ordered List Defines a ordered list, a list of items that have a specific order.
<li> List Item Dedines a list of items within an ordered or unordered list.
<figure> Figure Defines a unit of content, such as image, diagram, or code snippet, that is referenced in the main content.
<figcaption> Figure Caption Defines a caption or description of a <figure> element.
<blockquote> Blockquote Defines a section that is quoted from another source.
<cite> Citation Defines the title of a work, such as a book or a movie, that is being cited.
<time> Time Defines a specific time or a range of time.
<abbr> Abbreviation Defines a abbreviation or acronym.
<a> Anchor Defines a hyperlink, linking to another document or resource.
